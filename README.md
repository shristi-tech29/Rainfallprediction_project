# 🌧️ Rainfall Prediction Project

## 📌 Overview

The Rainfall Prediction Project is a Machine Learning application that predicts whether rainfall is likely to occur based on weather-related parameters. It uses a trained prediction model and provides an easy-to-use interface for users to enter data and receive predictions.

This project demonstrates the practical use of data science and machine learning techniques for weather forecasting and decision-making.

---

## 🚀 Features

* Predicts the possibility of rainfall using machine learning.
* User-friendly Python application.
* Interactive interface for entering weather parameters.
* Trained prediction model stored as a `.pkl` file.
* Includes Jupyter Notebook for model training and analysis.
* Generates quick and accurate predictions.

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/shristi-tech29/Rainfallprediction_project.git
```

2. Navigate to the project folder

```bash
cd Rainfallprediction_project
```

3. Install the required libraries

```bash
pip install pandas numpy scikit-learn
```

4. Run the application

```bash
python rain_app.py
```
